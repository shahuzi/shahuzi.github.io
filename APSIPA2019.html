End-to-End Emotional Speech Synthesis Using Style Tokens and Semi-Supervised Training

Paper:

http://arxiv.org/abs (Submitted to Asia-Pacific Signal and Information Processing Association Annual Summit and Conference 2019)

Authors:

Peng-fei Wu, Zhen-Hua Ling, Li-juan Liu, Hong-chuan Wu, Yuan Jiang, Li-Rong Dai

Abstract:

This paper proposed an end-to-end emotional speech synthesis (ESS) approach called semi-GST which uses global style tokens (GSTs) and semi-supervised training, it can implement ESS with a few number of emotion labels and maintain similar performance as a full-label-used model. Semi-GST defines a bank of tokens with the same number of emotion categories, and uses partial emotion labels to train the tokens so that the learned tokens have a good correlation with emotions. This method increases the token interpretability in GSTs model and can be used for automatic emotion annotation. Objective and subjective experimental results show that semi-GST outperformed a Tacotron-based semi-supervised ESS model when using the same number of emotion labels. In addition, it achieved similar performance with a Tacotron-based fully supervised training ESS model, but used only 5% emotion labels.
